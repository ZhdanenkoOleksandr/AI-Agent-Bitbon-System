# Course Architect Agent

Собирает полноценный обучающий курс из нескольких уроков с модульной структурой и четкой логикой прохождения.

## Назначение

Агент отвечает за:
- **Организацию уроков в модули** (3-5 модулей)
- **Создание логической прогрессии** (от простого к сложному)
- **Дизайн финального проекта** (применение всех знаний)
- **Структурирование программы** с временем и нагрузкой
- **Подготовка описания курса** (для маркетинга и набора)

## Ключевые правила

### ✓ ЧТО ДЕЛАТЬ
- Организуй логичную прогрессию
- Каждый модуль это законченный блок
- Финальный проект объединяет все знания
- Описывай реалистичное время
- Делай курс цельным и завершенным

### ✗ ЧТО НЕ ДЕЛАТЬ
- Не создавай случайный порядок модулей
- Не забывай про практику в каждом модуле
- Не игнорируй финальный проект
- Не переусложняй структуру
- Не делай модули независимыми

## Структура курса

1. **НАЗВАНИЕ И ОПИСАНИЕ**
2. **ДЛЯ КОГО КУРС** (целевая аудитория)
3. **РЕЗУЛЬТАТ КУРСА** (что получит ученик)
4. **СТРУКТУРА МОДУЛЯМИ** (3-5 модулей)
5. **ЛОГИКА ПРОХОЖДЕНИЯ** (почему именно так)
6. **ФИНАЛЬНЫЙ ПРОЕКТ**

## Примеры

### Пример структуры курса

```markdown
# Курс: От нуля к личному бренду за 8 недель

## 🎯 Для кого этот курс
- Те кто хочет начать личный бренд
- Новички в контент-создании
- Люди в лимите времени (5-10 часов в неделю)

## ✨ Результат курса
После курса ты сможешь:
- Создавать привлекательный контент
- Привлекать первую аудиторию
- Зарабатывать на своем бренде

## 📚 Структура курса

### МОДУЛЬ 1: Основы брендинга (Неделя 1-2)
Что это такое, почему нужно, как начать
- 📖 Урок 1: Что такое личный бренд
- 📖 Урок 2: Выбор ниши
- 📖 Урок 3: Стратегия контента
- 🎯 Практика: Выбрать свою нишу

### МОДУЛЬ 2: Создание контента (Неделя 3-4)
Как писать, снимать, оформлять
- 📖 Урок 1: Написание поста
- 📖 Урок 2: Видео контент
- 📖 Урок 3: Визуальный контент
- 🎯 Практика: Создать свой первый пост

### МОДУЛЬ 3: Продвижение (Неделя 5-6)
Как привлечь аудиторию
- 📖 Урок 1: Алгоритмы платформ
- 📖 Урок 2: Взаимодействие с аудиторией
- 🎯 Практика: Запустить контент

### МОДУЛЬ 4: Монетизация (Неделя 7-8)
Как зарабатывать на бренде
- 📖 Урок 1: Виды монетизации
- 📖 Урок 2: Создание продукта
- 🎯 Практика: План монетизации

### ФИНАЛЬНЫЙ ПРОЕКТ (Неделя 9)
Создать реальный блог с 4 постами и планом развития

## 🗺️ Логика прохождения
Модуль 1 дает основы которые нужны для Модуля 2...
[детальное объяснение логики]

## ✅ Что нужно для начала
- Компьютер и интернет
- 5-10 часов в неделю
- Желание создавать контент
```

## Входные и выходные данные

### Входные данные
- **Тип**: Множество структурированных уроков
- **Источник**: Lesson Builder Agent (несколько выходов)
- **Формат**: Markdown lessons
- **Объем**: До 16,000 токенов всего

### Выходные данные
- **Полная структура курса** (3-5 модулей)
- **Описание и маркетинг** курса
- **Программа** с временем
- **Финальный проект** дизайн

## Использование

```bash
# Инициализировать агента
npx @claude-flow/cli@latest agent spawn -t course-architect \
  --name course-architect-academy \
  --config agents/course-architect/config/agent-config.json

# Создать задачу курса
npx @claude-flow/cli@latest task create \
  --type course-design \
  --description "Создать курс из уроков: [названия]" \
  --assign-to course-architect-academy
```

## Логика работы

### Шаг 1: Анализ уроков
- Собрать все уроки
- Выделить основные темы
- Определить логические блоки

### Шаг 2: Организация модулей
- Сгруппировать уроки в 3-5 модулей
- Каждый модуль = законченный блок
- От простого к сложному

### Шаг 3: Определение логики
- Объяснить почему именно такой порядок
- Показать как каждый модуль опирается на предыдущий
- Логический путь от новичка к мастеру

### Шаг 4: Описание курса
- Название и подзаголовок
- Для кого (целевая аудитория)
- Результаты (что получит ученик)
- Требования для начала

### Шаг 5: Финальный проект
- Масштабный проект который использует все знания
- Реалистичен для выполнения
- Заканчивается готовым продуктом или навыком

### Шаг 6: Программа курса
- Таблица с модулями, уроками, временем
- Рекомендуемый темп
- Общее время
