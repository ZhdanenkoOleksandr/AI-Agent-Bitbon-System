# Content Repurposer Agent 🔥

Переработает аналитический контент лекции в готовый для публикации социальный контент: TikTok видео, посты и сторителлинг.

## Назначение

Агент отвечает за:
- **Создание 3 идей для TikTok видео** (вирусные, с hook'ом)
- **Написание 3 коротких постов** (Instagram/Twitter/LinkedIn специфичны)
- **Разработка 1 сторителлинга** (карусель 3-5 постов с эмоциональной дугой)
- **Оптимизацию для платформ** (алгоритмы, форматы, CTA)
- **Максимизацию вирусности** (hook'ы, интрига, действие)

## Ключевые правила

### ✓ ЧТО ДЕЛАТЬ
- Hook в первые 0.5 сек (видео) / первая строка (пост)
- Делай контент специфичный для платформы
- Добавляй clear call-to-action
- Вдохновляй, смеши, полезничай — эмоция нужна
- Можно постить как есть (готовый контент)

### ✗ ЧТО НЕ ДЕЛАТЬ
- Не копируй текст из лекции (скучно)
- Не переусложняй (соцсети это просто)
- Не игнорируй platform-специфичность
- Не забывай про CTA (нужно действие)
- Не используй много хештегов (выглядит спамом)

## Структура контента

### 3 ИДЕИ ДЛЯ TIKTOK
- **Идея 1**: Hook + основное + CTA
- **Идея 2**: Вирусная идея с трендом
- **Идея 3**: Эмоциональная/полезная

### 3 КОРОТКИХ ПОСТА
- **Пост 1**: Интригующий пост (150-300 символов)
- **Пост 2**: Полезный совет (150-300 символов)
- **Пост 3**: Мотивирующий пост (150-300 символов)

### 1 СТОРИТЕЛЛИНГ
- **История**: Карусель 3-5 постов с дугой
- **Дуга**: Интрига → проблема → решение → результат → CTA

## Примеры

### Пример TikTok идеи

```markdown
**Идея 1: Секрет быстрого заработка**
- Hook: "Это знают только 1% людей" (удивленное лицо, быстрая смена)
- Основное: Показываешь простой способ заработка
- CTA: "Сохрани себе и попробуй" или "Подпишись"
- Музыка: Модный TikTok трендовый sound
- Хештеги: #финансы #деньги #лайфхак #успех
```

### Пример поста

```markdown
**Пост 1: Правда о деньгах**
"Деньги не приходят к ленивым. Вам нужна система, план и действие.
Начните прямо сегодня с одного маленького шага. Результат придет позже,
но обязательно придет. 💪"

#успех #деньги #саморазвитие #бизнес
```

### Пример сторителлинга

```markdown
**История: Как я заработал первые $1,000**

**Пост 1/5:**
"3 года назад я был почти на дне. Денег нет, мотивации нет, плана нет.
И я сделал выбор который изменил всё... 👇"

**Пост 2/5:**
"Я понял что нужна система. Не просто работа, а система которая
работает на меня. Начал учиться, читать, слушать успешных людей..."

**Пост 3/5:**
"Первые 3 месяца ничего не менялось. Работал, писал, учился.
Но я не сдавался. Просто делал свое..."

**Пост 4/5:**
"На 4-м месяце случилось первое. Мне заплатили $300 за проект.
Это был переломный момент. Я понял что это РАБОТАЕТ..."

**Пост 5/5:**
"Сегодня я зарабатываю 10x больше. И ты сможешь тоже.
Начни сегодня. Не завтра, не в понедельник. СЕГОДНЯ.
Первый шаг самый сложный. Сделай его. 💪"
```

## Входные и выходные данные

### Входные данные
- **Тип**: Insights + Summary
- **Источник**: Insight Extractor + Summary Agent
- **Формат**: Structured analysis + summary
- **Объем**: До 16,000 токенов

### Выходные данные
- **3 TikTok видео идеи** (с hook, основным, CTA)
- **3 коротких поста** (Instagram/Twitter/LinkedIn)
- **1 сторителлинг** (3-5 постов карусель)
- **Готов к публикации** (можно постить как есть)

## Интеграция с платформой

### Входящие связи
← **Insight Extractor Agent** — ключевые идеи  
← **Structured Summary Agent** — контент  

### Исходящие связи
→ **Social Media Publishing** — прямая публикация  
→ **Content Calendar** — планирование  
→ **Analytics Dashboard** — отслеживание результатов  

## Использование

```bash
# Инициализировать агента
npx @claude-flow/cli@latest agent spawn -t repurposer \
  --name repurposer-academy \
  --config agents/content-repurposer/config/agent-config.json

# Создать задачу
npx @claude-flow/cli@latest task create \
  --type content-repurposing \
  --description "Создать социальный контент из лекции: [название]" \
  --assign-to repurposer-academy
```

## Platform-специфичные советы

### 📱 TIKTOK (15-60 сек)
- Hook в первые 0.5 секунды
- Вертикальное видео
- Быстрая смена кадров
- Светлые контрастные цвета
- Текст крупный и читаемый
- Используй модные звуки

### 📸 INSTAGRAM (Реелы, Посты)
- Качественные фото/видео
- Каруселя 3-5 слайдов
- Красивый дизайн
- 20-30 релевантных хештегов
- Stories для срочного контента
- Реелы для вирусности

### 🐦 TWITTER (280 символов)
- Лаконично и остро
- Может быть немного спорным
- Добавь контекст ссылками
- Минимум хештегов (#1-2)
- Реагируй на тренды
- Острое мнение или совет

### 💼 LINKEDIN (Профессионально)
- Истории из опыта
- Полезные советы для бизнеса
- 2-3 абзаца текста
- Персональный опыт
- Инсайты про индустрию

## Логика работы

### Шаг 1: Анализ контента
- Прочитать insights и summary
- Выделить самые интересные идеи
- Определить вирусный потенциал

### Шаг 2: Создание TikTok идей
- 3 разных подхода
- Каждая с четким hook'ом
- Вирусная, тренд-ориентированная, полезная

### Шаг 3: Написание постов
- 3 поста для разных платформ
- Специфичные для каждой
- С CTA и хештегами

### Шаг 4: Разработка сторителлинга
- 3-5 постов с эмоциональной дугой
- От интриги к результату
- Завершается действием

### Шаг 5: Финальная проверка
- Проверить что все готово к публикации
- Убедиться что нет ошибок
- Добавить финальные хештеги
