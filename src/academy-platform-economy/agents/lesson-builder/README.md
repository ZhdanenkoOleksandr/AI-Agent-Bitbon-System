# Lesson Builder Agent

Создает полноценный, готовый к преподаванию урок на основе конспекта с примерами и заданиями.

## Назначение

Агент отвечает за:
- **Структурирование урока** с 6 логичными секциями
- **Расширение контента** с примерами и объяснениями
- **Создание практических заданий** для немедленного применения
- **Дизайн домашней работы** для закрепления знания
- **Подготовка к следующему уроку** (логическая цепь)

## Ключевые правила

### ✓ ЧТО ДЕЛАТЬ
- Пиши для новичка без фоновых знаний
- Добавляй 2-3 практических примера
- Делай задания выполнимыми за 15-30 минут
- Структурируй информацию с заголовками
- Проверяй что все понятно

### ✗ ЧТО НЕ ДЕЛАТЬ
- Не пиши сложный академический текст
- Не игнорируй практику
- Не создавай задания которые сложнее урока
- Не пропускай примеры
- Не делай скучный материал

## Структура урока

1. **НАЗВАНИЕ** (Ясное название + время)
2. **ВВЕДЕНИЕ** (Почему это важно)
3. **ОСНОВНОЙ МАТЕРИАЛ** (Разбитый на блоки)
4. **ПРИМЕРЫ** (2-4 практических примера)
5. **ПРАКТИЧЕСКОЕ ЗАДАНИЕ** (Прямо в уроке)
6. **ДОМАШНЕЕ ЗАДАНИЕ** (На неделю)

## Использование

```bash
# Инициализировать агента
npx @claude-flow/cli@latest agent spawn -t lesson-builder \
  --name lesson-builder-academy \
  --config agents/lesson-builder/config/agent-config.json

# Создать задачу урока
npx @claude-flow/cli@latest task create \
  --type lesson-creation \
  --description "Создать урок на основе: [название]" \
  --assign-to lesson-builder-academy
```

## Логика работы

### Шаг 1: Анализ summary
- Прочитать конспект целиком
- Понять основные блоки
- Выделить ключевые мысли

### Шаг 2: Написание введения
- Почему этот урок важен
- Что конкретно научишься
- Как это изменит твою жизнь

### Шаг 3: Структурирование материала
- Разделить на 3-5 логичных блоков
- Каждый блок = одна идея
- Логичные переходы между блоками

### Шаг 4: Добавление примеров
- 2-4 практических примера
- Из реальной жизни если возможно
- Наглядные и запоминаемые

### Шаг 5: Практическое задание
- Упражнение на 5-15 минут
- Можно выполнить прямо сейчас
- С критериями успеха

### Шаг 6: Домашнее задание
- Задание на 1-7 дней
- Применяет все из урока
- Четкие критерии успеха
