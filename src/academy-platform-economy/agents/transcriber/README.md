# Transcriber Agent

Специализированный агент для преобразования аудио и видео материалов в точный текст для платформы ACADEMY PLATFORM ECONOMY.

## Назначение

Агент отвечает за:
- **Транскрипцию аудио** в текст без искажений
- **Транскрипцию видео** с сохранением контекста
- **Подтверждение существующего текста** (если уже есть готовая транскрипция)
- **Структурирование контента** по смыслу и логике
- **Сохранение авторского стиля** и структуры речи

## Ключевые правила

### ✓ ЧТО ДЕЛАТЬ
- Записывай каждое слово полностью
- Разделяй текст на абзацы по смыслу
- Сохраняй структуру оригинальной речи
- Обозначай переходы между темами
- Отмечай непонятные фрагменты как `[неразборчиво]`

### ✗ ЧТО НЕ ДЕЛАТЬ
- Не сокращай слова
- Не улучшай грамматику
- Не переформулируй предложения
- Не добавляй свои примеры
- Не интерпретируй смысл

## Интеграция с платформой

Агент готов к использованию в следующих сценариях:

1. **Импорт нового аудио/видео контента**
   - Загрузка файла → Transcriber Agent → Транскрипция → Сохранение

2. **Проверка существующего контента**
   - Готовый текст → Transcriber Agent → Подтверждение → Публикация

3. **Обновление контента**
   - Переснятая лекция → Transcriber Agent → Новая транскрипция → Замена

## Использование

```bash
# Инициализировать агента в проекте
npx @claude-flow/cli@latest agent spawn -t transcriber \
  --name transcriber-academy \
  --config src/academy-platform-economy/agents/transcriber/config/agent-config.json

# Отправить задачу агенту
npx @claude-flow/cli@latest task create \
  --type transcription \
  --description "Транскрибировать лекцию: Introduction to Blockchain" \
  --assign-to transcriber-academy
```

## Метаданные агента

- **Роль**: specialist
- **Модель**: claude-opus-4-6
- **Входные форматы**: mp3, wav, m4a, ogg, mp4, webm, mov, avi, txt
- **Выходной формат**: markdown (структурированный текст)
- **Максимум токенов**: 16000
- **Температура**: 0 (максимальная точность)
