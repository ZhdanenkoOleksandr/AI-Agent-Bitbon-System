## 🎯 Агенты в действии

### Этап 1: Входящий контент → Текст

#### [1] Transcriber Agent ✅
- **Входы**: Аудио (mp3, wav, m4a, ogg) или Видео (mp4, webm, mov, avi)
- **Задача**: Точная транскрипция с сохранением структуры
- **Выход**: Сырой текст лекции
- **Время**: 3-5 мин (для часовой лекции)
- **Качество**: 95%+

### Этап 2: Текст → Чистый текст

#### [2] Cleaner Agent ✅
- **Входы**: Сырой текст от Transcriber
- **Задача**: Удалить паразиты, исправить ошибки, структурировать
- **Выход**: Чистый, структурированный текст
- **Время**: 1-2 мин
- **Результат**: Готов к анализу

### Этап 3: Анализ контента

#### [3] Insight Extractor Agent ✅
- **Входы**: Чистый текст
- **Задача**: Извлечь суть лекции
  - 10 ключевых идей
  - 5 главных инсайтов
  - 3 модели/подхода
  - Важные термины
- **Выход**: Структурированный анализ
- **Время**: 2-3 мин

### Этап 4: Конспект для учебы

#### [4] Structured Summary Agent ✅
- **Входы**: Insights + Clean Text
- **Задача**: Создать понятный конспект
- **Структура**: 
  1. О чем лекция
  2. Основные идеи (кратко)
  3. Подробное объяснение
  4. Примеры
  5. Выводы
- **Выход**: Готовый конспект для чтения
- **Время**: 2-3 мин

### Этап 5: Персонализация 🔥

#### [5] Personalization Agent ✅
- **Входы**: Insights + Summary
- **Задача**: Адаптировать под пользователя (путь к $1M через Bitbon)
- **Выход**:
  1. Как применить лично
  2. Конкретные действия (от сегодня до квартала)
  3. Связь с Bitbon системой
  4. Идеи для проектов (с потенциалом прибыли)
- **Время**: 3-5 мин
- **Ценность**: Максимальная для целей пользователя

### Этап 6: Полноценный урок

#### [6] Lesson Builder Agent ✅
- **Входы**: Summary (Structured Summary)
- **Задача**: Создать структурированный урок
- **Структура**:
  1. Название урока
  2. Введение
  3. Основной материал (с подразделами)
  4. Примеры
  5. Практическое задание
  6. Домашнее задание
- **Выход**: Урок готов к преподаванию
- **Время**: 3-5 мин
- **Использование**: Может быть модулем в курсе

### Этап 7: Курс из уроков

#### [7] Course Architect Agent ✅
- **Входы**: Множество уроков
- **Задача**: Собрать полный курс
- **Структура**:
  1. Название курса + описание
  2. Для кого курс
  3. Результат курса
  4. Модульная структура (3-5 модулей)
  5. Логика прохождения
- **Выход**: Полный курс с программой
- **Время**: 5-10 мин (зависит от кол-ва уроков)

### Этап 8: Контент для социалок 🔥

#### [8] Content Repurposer Agent ✅
- **Входы**: Insights + Summary
- **Задача**: Переработать контент для социалок
- **Выход**:
  1. **3 идеи для TikTok** (с hook, основным, CTA)
  2. **3 коротких поста** (Instagram/Twitter/LinkedIn)
  3. **1 сильный сторителлинг** (карусель 3-5 постов)
- **Время**: 2-3 мин
- **Ценность**: Готовый контент, который можно постить
- **Виральность**: Оптимизировано для максимального вовлечения

### Этап 9: QA Проверка (Планируется) 📋

#### [9] QA Agent
- **Входы**: Контент перед публикацией
- **Задача**: Проверить качество
  - Нет ошибок
  - Соответствие стандартам
  - Полнота информации
  - Правильность форматирования
- **Выход**: Одобрено или замечания для исправления

### Этап 10: Публикация (Планируется) 📋

#### [10] Publisher Agent
- **Входы**: Проверенный контент
- **Задача**: Опубликовать на всех каналах
  - Платформа (сайт, LMS, CMS)
  - Социальные сети
  - Email подписчикам
  - Архив
- **Выход**: Контент доступен везде

## 🚀 Пример использования

### Сценарий: Создать полный контент пакет из одной лекции

```bash
# 1. Транскрибировать аудио
npx @claude-flow/cli@latest task create \
  --type transcription \
  --description "Транскрибировать лекцию: blockchain-basics.mp3" \
  --assign-to transcriber-academy

# 2-8. Запустить все агенты в конвейере

  --template full-content-pipeline \
  --input-file lecture-transcript.md
```
